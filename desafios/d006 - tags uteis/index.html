Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Parágrafo
  • Título
  • Destaque / Negrito
  • Quebra de linha
  • Lista ordenada / Order List
  • Lista não ordenada / Unorder List
  • Comentário
  • Imagens
  • Código-fonte
  • Abreviações
  • Estrutura básica HMTL
  • Citação simples
  • Formatação em codigo
  • Formatação de texto (mostra o texto conforme foi escrito, com espacos e tudo mais)
